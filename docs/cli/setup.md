# `openhand setup`

Initialize `~/.openhand/openhand.json` and the agent workspace.

Related:

- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
openhand setup
openhand setup --workspace ~/.openhand/workspace
```

To run the wizard via setup:

```bash
openhand setup --wizard
```
